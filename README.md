Long Description (for README top section)

ai-cli-tools is an all-in-one AI command-line utility designed for speed, simplicity, and real productivity. Everything works directly from your terminal — no UI bloat, no complicated setup.

This toolkit gives you:

AI Summarizer → Summarize .txt or .pdf files into short, medium, or long formats

MCQ Generator → Turn any text or PDF into quizzes with answer keys

PNG → WEBP Converter → Convert single files or entire folders

Bulk Smart Renamer → AI-like intelligent naming patterns for messy files

YouTube Transcript Summarizer → Extract + summarize videos instantly

Perfect for students, developers, creators, write-up makers, and anyone who wants quick AI tools without opening a browser.
